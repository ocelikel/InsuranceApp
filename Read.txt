Projede kullanýlan teknolojiler ve tasarým kalýplarý

.net 5
MassTransit/RabbitMq/Saga State Machine/Redis
SignalR
CQRS pattern
Asp.Net Core MVC
Asp.Net Web Api

-Projede client tarafýndan bilgiler alýnýp backendde asenkron olarak sigorta firmalarýna iletilir.
Teklifler hazýrlandýðýnda clienta SignalR ile iletilir.
-Alternatif olarak Signalr yerine clienta teklif iletimi için ajax poll altyapýsý için backendde apiler yazýlmýþ ve saga state machine configürasyonu yapýlmýþtýr.

-Teklif için sorgulama yapýldýðýnda asenkronluðu göstermek için server tarafýnda 15sn'lik bir Delay eklenmiþtir.

Projeyi kaldýrmak için ilk olarak dockerda rabbitmq ve redis ayaða kaldýrýlmalý.
